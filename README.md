
# پروژه رستوران تمرینی 

این پروژه توسط بنده به وسیله جیسون سرور در گیت هاب پیج ران شده و تصاویر از یک هاست دیگه میاد 


## Deployment

To deploy this project run

```bash
  npm start
```


## 🛠 موارد استفاده شده
React , pure css no library
